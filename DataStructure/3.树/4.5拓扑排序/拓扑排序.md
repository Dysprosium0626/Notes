# 拓扑排序

AVO网络 Activity On Vertex


拓扑序：如果图中从V到W有一条有向路径的话，则V一定排在W之前。满足次条件的顶点序列称为一个拓扑序

获得一个拓扑序的过程就是拓扑排序

AOV如果有合理的拓扑序，则必定是有向无环图Directed Acyclic Graph DAG

不存在环，V必须在V之前结束



抹掉顶点，抹掉边！


每次选择入度为0的顶点


```c
void TopSort();
```



图中有回路ERROR


聪明的算法：入度为0的顶点放在容器里，下次直接在容器里找度为0的顶点就行
